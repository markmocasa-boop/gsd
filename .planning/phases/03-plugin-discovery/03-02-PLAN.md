Add a showPluginInfo(pluginName) function that:
1. Validates pluginName is provided (error if missing)
2. Builds path: ~/.claude/{pluginName}/plugin.json
3. Checks if plugin exists (error with "Plugin not installed" if missing)
4. Parses manifest, handles JSON parse errors gracefully

Display formatted output:
```
Plugin: {name} v{version}
{description}

Author: {author}
{If repository: Repository: {repository}}

{If gsd.commands:}
Commands:
  /{command.name} - {command.description}
  ...

{If gsd.agents:}
Agents:
  {agent.name} - {agent.description}
  ...

{If gsd.hooks:}
Hooks:
  {hook.event} - {hook.description}
  ...

Installation:
  Installed: {_installed.date formatted as YYYY-MM-DD}
  {If linked: Linked from: {_installed.source}}
  {If not linked: Location: ~/.claude/{name}/}
```

Use cyan for plugin name, dim for descriptions, green for checkmarks where appropriate.
